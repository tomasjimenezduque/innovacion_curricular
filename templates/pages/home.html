CRUD - Base de Datos Facturas

Bienvenido al frontend Flask que consume la API REST de FastAPI.

Este proyecto es un tutorial paso a paso para aprender:

  • Flask como framework web frontend
  • Consumo de API REST con la librería requests
  • Plantillas HTML con Jinja2 (herencia y componentes)
  • Bootstrap 5 para estilos responsivos
  • Patrón de servicio genérico (ApiService)
  • Operaciones CRUD completas (Crear, Leer, Actualizar, Eliminar)
Tablas disponibles:
  • Producto — Gestión completa de productos (código, nombre, stock, valor)
API Backend: La API FastAPI debe estar corriendo en http://localhost:8000 para que este frontend funcione correctamente.
Ejecuta en la carpeta de la API: uvicorn main:app --reload
